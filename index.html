DOKUMENTASI KINERJA A1

Konfigurasi Security & Access Gateway

Membangun gerbang nirkabel aman berbasis RouterOS. Mengatur mode **Station**, memindai SSID eksternal, menetapkan enkripsi **WPA2-PSK**, serta mengaktifkan **DHCP Client**, **DNS Server**, dan **NAT Masquerade** agar internal client terhubung ke internet global.

Perangkat Target:RouterBOARD RB951-2nd
Keamanan Utama:WPA2-PSK Enkripsi
Layanan Jaringan:NAT Masquerade & DNS Relay

Skrip disalin! Siap diunggah ke Winbox CLI.

DOKUMENTASI KINERJA A2

Alat Ukur Detak Jantung / Heart Rate

Merakit sensor analog detak jantung berbasis optik inframerah dengan mikrokontroler **Arduino UNO**. Sinyal listrik analog dari denyut darah diproses menggunakan sampling frekuensi 60Hz, menyaring derau, dan mengirimkan paket serial telemetri ke konsol visualizer.

Microcontroller:Arduino UNO R3 ATmega328P
Sensor Fisik:Analog Optical Heart Pulse Sensor
Kanal Telemetri:Serial USART Stream (9600 bps)

Source code disalin! Siap dijalankan di Arduino IDE.

Serial Oskiloskop Telemetri Detak Jantung
Aktif: 75 BPM SIGNAL STABLE
Sketsa Arduino IDE (.ino) 
#define PULSE_PIN A0
int signalValue = 0;

void setup() {
  Serial.begin(9600);
  pinMode(PULSE_PIN, INPUT);
}

void loop() {
  signalValue = analogRead(PULSE_PIN);
  Serial.print("TELEMETRY_RAW:");
  Serial.println(signalValue);
  delay(15); // Sampling Rate 60Hz
}
DOKUMENTASI KINERJA A3

HTTP Server & Cloudflare Tunnel

Mempublikasikan situs web lokal ke ranah publik tanpa menyewa IP Publik Statis. Server web dijalankan secara lokal via **Python HTTP Engine** (Port 8000), lalu dihubungkan secara instan melalui daemon agen **Cloudflared** ke jaringan global Cloudflare terenkripsi SSL/TLS.

Server Lokal:Python 3 SimpleHTTPServer
Secure Tunnel:Cloudflare Daemon (Cloudflared)
Protokol Enkripsi:TLS v1.3 Secure Handshake

Konfigurasi YAML berhasil disalin!
